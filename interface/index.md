# Interface

The editing surface for [[graph_data/index|Graph Data]]:

- A visual graph editor/viewer built in **Godot**.
- A **markdown** round-trip layer -- human-editable notes that sync up and
  down with the graph (this vault doubling as a subgraph view, courtesy of
  Obsidian's links and graph view).

## Design tasks

- Decide the markdown <-> graph round-trip format (what frontmatter/links
  map to which node/edge types).
- Scope the Godot front-end: what views are needed (character browser,
  appearance editor, class/ability editor)?
